# ORNG 90 DAY EXECUTION PLAN

## DAYS 1 TO 30. FOUNDATION AND DATA.
**Daily:**
- Study one math concept.
- Write one notebook.
- Log one insight.

**Weekly goals:**
- Probability and statistics refresh.
- Time series basics.
- Data cleaning pipeline built.
- First clean equity dataset ready.

**Deliverables:**
- Clean data store.
- Data audit report.
- Baseline metrics notebook.

## DAYS 31 TO 60. FACTORS AND STRATEGIES.
**Daily:**
- Build one factor.
- Test one parameter set.
- Kill weak logic fast.

**Weekly goals:**
- Momentum library.
- Mean reversion library.
- Volatility factor tests.
- Two baseline strategies.

**Deliverables:**
- Factor performance tables.
- Strategy blueprints.
- Failure logs.

## DAYS 61 TO 90. VALIDATION AND RISK.
**Daily:**
- Break your own strategies.
- Stress test assumptions.
- Rewrite slow code.

**Weekly goals:**
- Walk forward framework.
- Monte Carlo tests.
- Risk rules integration.
- Strategy ranking system.

**Deliverables:**
- Robustness reports.
- Risk attribution sheets.
- Final research summary.
