# ORNG DAILY OPERATING LOOP

**Every day:**
1. **Hypothesis.** Define a clear, testable idea.
2. **Test.** Run the pipeline on historical data.
3. **Measure.** Check Sharpe, Drawdown, and Stability.
4. **Decide.** Keep, Refine, or Kill.
5. **Log.** Record the result in `archive/daily_logs`.

**No exceptions.**

## DAILY OPERATING CHECKLIST
**Every run:**
- Update data.
- Rebuild factors.
- Re run baseline.
- Compare metrics.
- Log outcome.
